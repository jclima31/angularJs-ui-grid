GridOptions
object in module ui.grid.pinning

Description

GridOptions for pinning feature, these are available to be set using the ui-grid gridOptions

Properties

  • enablePinning

    Enable pinning for the entire grid.
    Defaults to true

  • hidePinLeft

    Hide Pin Left for the entire grid.
    Defaults to false

  • hidePinRight

    Hide Pin Right pinning for the entire grid.
    Defaults to false
